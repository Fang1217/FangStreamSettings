On the Subject of Wires

Wires are the lifeblood of electronics! Wait, no, electricity is the lifeblood. Wires are more like the arteries. The veins? No matter…

  • A wire module can have 3-6 wires on it.
  • Only the one correct wire needs to be cut to disarm the module.
  • Wire ordering begins with the first on the top.
3 wires:
If the last wire is black and the serial number starts with a letter, cut the first wire.
Otherwise, If there is exactly one black wire, cut the second wire.
Otherwise, If there is more than one black wire, cut the last wire.
Otherwise, cut the first wire.
4 wires:
If there is exactly one black wire, cut the third wire.
Otherwise, If there is more than one yellow wire, cut the last wire.
Otherwise, If there are no white wires, cut the second wire.
Otherwise, If the last wire is black, cut the first wire.
Otherwise, cut the second wire.
5 wires:
If the last wire is white and there is a DVI-D port present on the bomb, cut the fourth wire.
Otherwise, If there is exactly one black wire and the last digit of the serial number is odd, cut the second wire.
Otherwise, If there is more than one yellow wire and there is a parallel port present on the bomb, cut the last wire.
Otherwise, cut the fourth wire.
6 wires:
If the last wire is yellow and the last digit of the serial number is odd, cut the second wire.
Otherwise, If there is more than one black wire, cut the last black wire.
Otherwise, If there is more than one black wire, cut the first wire.
Otherwise, cut the fifth wire.
